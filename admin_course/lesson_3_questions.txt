Length of String: Write a program that calculates and prints the length of a given string without using the built-in len() function.
# Example Input: 
n = "hello"
# Example Output:
5
# Example Input: 
n = "dog"
# Example Output:
3

Vowel Counter: Given a string, write a program to count and display the number of vowels (both lowercase and uppercase) in the string. (vowel is when character is: 'a, e, i, o, u')
# Example Input: 
n = "hello"
# Example Output:
3

Number of Words: Write a program that counts and displays the number of words in a given sentence.
# Example Input: 
n = "hello my friend"
# Example Output:
3
# Example Input: 
n = "john went to the park"
# Example Output:
5

Letter Capitalizer: Create a program that capitalizes the first letter of a given sentence.
# Example Input: 
n = "hello my friend"
# Example Output:
"Hello My Friend"
# Example Input: 
n = "john went to the park"
# Example Output:
"John Went To The Park"

Email Address Validator: Design a program that validates whether a given string is a valid email address. ONLY consider gmail addresses, all other addresses will be invalid. (first step is to think through what an email address contains, and what you need to check for.). 
# Example Input: 
n = "hello@gmail.com"
# Example Output:
"Valid"
# Example Input: 
n = "hello@"
# Example Output:
"Not Valid"
# Example Input: 
n = "hello"
# Example Output:
"Not Valid"
# Example Input: 
n = "hello@gmail"
# Example Output:
"Not Valid"
