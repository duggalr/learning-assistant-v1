Problem

Students do not have access to a personalized, 1-to-1 educational system. There is no scalable solution that exists, to allow a personalized 1-to-1 education for each student.

It should come as no surprise that offering a more personalized education approach is a more optimal medium for fostering student learning, compared to the generalized education system we currently rely on.

For students, some of the advantages can include:

  • Develop a greater understanding of the material.
  • Progress at their own pace, going on their own learning tangents.
  • Develop greater intrinsic motivation for the material they are learning, and learning more generally.
  • Develop greater confidence in themselves, feeling less pressured by external factors.

We are at a pivotal point in technology, where we now have a general AI system (GPT-4) which can be used to perform many different tasks, in a (mostly) reliable manner.

I believe Code Companion, which leverages GPT-4 (as described below), is the first step towards creating a more personalized educational system for students.

AI, nor software alone will enabled personalized education. Teachers will always play a pivotal role.

However, the current AI systems do enable a new paradigm of software, which can be leveraged in the classroom, to serve as a strong medium to help create a more personalized education environment for each student.

If you are interested in helping solve this problem, feel free to reach out.

Our Approach

Watch the Demo.

As an initial solution to the problem described above, I built Code Companion, a GPT-4 personalized tutor designed to help students with programming-specific questions or difficulties.

Like a good tutor, Code Companion will not give you the answer. Instead, just like a great tutor, it will help guide you in the right direction by asking you questions.

It will force you to think through the problem so you can figure out the solution on your own.

Currently with Code Companion, you can:

  • Utilize our In-Built Editor & Interpreter, where you can work on the question, and ask Code Companion for help at every step along the way.
  • Ask Code Companion to generate thorough explanations and examples for specific concepts.
  • Ask Code Companion to generate questions to help reinforce your understanding of specific concepts.
  • Ask Code Companion to provide feedback, or a potential new way to think about a problem.
  • We have in-built Python exercises, for you to experience/use the AI, while gaining foundational programming knowledge.
