About:

Code Companion is a GPT-4 based assistant, that helps answer any programming questions or difficulties you run into. Simply write your code in our online editor (currently support Python), and ask any questions you have to our assistant.

Code Companion is not here to give you the answer. Instead, just like a great tutor, it will help guide you in the right direction by asking you questions. It will force you to think through the problem so you can figure out the solution on your own.

It is a great tool for those who are starting to learn programming, and want some additional assistance or guidance along the way!

You can use Code Companion to:

  • Ask Code Companion specific questions about your code or a bug you are facing.
  • Ask Code Companion to generate thorough explanations and examples for specific concepts.
  • Ask Code Companion to generate questions to help reinforce your understanding of specific concepts.

Potential Features down the line:

  • Generating a full study plan for learning the fundamentals of Python Programming.
  • Providing online practice tests and personalized learning plans, based on each individual's skills.
  • Creating a VSCode extension, so you can 'bring your tutor' everywhere, in your programming journey.

Reach out if you have ideas, suggestions, questions, or want to help engineer Code Companion!
